## About This Document

This document utilizes `h1`, `h2`, `p`, and `a` components for structuring and linking content.

You can find more information about these components in the [source code](https://github.com/Diogo-k/portfolio/blob/master/src/components/MDX/MDXMarkdown.js).

### Key Features

Our components offer several features to enhance your content:

* **Accessible Headings:** Headings (`h1` through `h6`) automatically generate anchor links for easy navigation.
* **Styled Links:** External links are clearly indicated.
* **Beautiful Typography:** Paragraphs are styled for readability.
